# rofi TokyoNight colors scheme by Rey

## Launch rofi

` rofi -theme reynight -show drun -modi run,drun -show-icons`

### if you use i3 wm, you could:

`bindsym $mod+d exec "rofi -theme reynight -show drun -modi run,drun -show-icons"`
